# 2019-nCoV API

> Coronavirus (2019-nCoV) API

Frontend can be found here [2019-ncov-frontend](https://github.com/sorxrob/2019-ncov-frontend).

## Build Setup

``` bash
# set consumer keys and port
$ cp .env.example .env

# install dependencies
$ npm install

# serve at localhost:3000
$ npm start

# start cron
$ node runner
```
